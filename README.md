# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, PyPDF2, and Google Gemini.

## Features

- Upload Resume PDF
- Extract Resume Text
- ATS Score
- Skill Analysis
- Missing Skills
- Resume Suggestions
- Career Recommendations
- Interview Questions

## Technologies

- Python
- Streamlit
- PyPDF2
- Google Gemini API

## Run
create a new api key 

pip install -r requirements.txt

streamlit run app.py
